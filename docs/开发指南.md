# 开发指南

## 环境准备

1. 安装 JDK 17、Maven 3.9+、Node.js 18 LTS、Docker Desktop
2. VS Code 安装推荐插件（见安装清单）
3. `git clone` 本项目

## 本地开发流程

### 1. 启动中间件
```bash
docker-compose -f saas-deploy/docker/docker-compose/docker-compose.base.yml up -d
```

### 2. 启动后端
```bash
cd saas-backend
mvn clean install -DskipTests
# 在IDE中启动各个服务，或使用 mvn spring-boot:run
```

### 3. 启动前端
```bash
# Admin (端口 5173)
cd saas-frontend-admin
npm install
npm run dev

# Mall (端口 5174)
cd saas-frontend-mall
npm install
npm run dev
```

## Claude Code Subagent 使用

使用自然语言触发对应 agent：
- 架构设计：描述需求，agent 会自动读取 context 并输出设计
- 后端开发：描述接口，agent 按分层生成代码
- 前端开发：描述页面，agent 按模板生成 Vue 页面
- 数据库设计：描述实体，agent 生成 DDL
- 测试：描述测试范围，agent 生成测试代码
- 审查：agent 自动 diff 并逐文件审查
- 调试：粘贴错误信息，agent 定位并修复
- 部署：描述环境，agent 生成 Docker/Jenkins 配置
