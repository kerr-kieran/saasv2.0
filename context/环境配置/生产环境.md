# 生产环境

## 服务器规划

| 角色 | 数量 | 规格 |
|------|------|------|
| Nginx | 1-2 | 2C4G |
| Gateway | 2+ | 2C4G |
| 业务服务 | 8x2=16 | 4C8G |
| MySQL Master | 1 | 8C16G, 500G SSD |
| MySQL Slave | 1-2 | 8C16G, 500G SSD |
| Redis Cluster | 3主3从=6 | 4C8G |
| RocketMQ | 1 NS + 2 Broker | 4C8G |
| Nacos | 2-3 | 2C4G |

## 高可用

- Nginx: Keepalived 双机热备
- Gateway: 至少2实例，Nginx upstream
- MySQL: 主从 + MHA自动切换
- Redis: Cluster 3主3从
- RocketMQ: Dledger 主从自动切换
- Nacos: 3节点集群

## 监控（后续接入）

- 链路追踪: SkyWalking
- 日志: ELK (Elasticsearch + Logstash + Kibana)
- 告警: AlertManager → 飞书/钉钉
