# 布局规范

## 后台管理布局

```
┌────────────────────────────────────┐
│  Logo      顶部导航栏    用户信息  │  Header (60px)
├──────┬─────────────────────────────┤
│      │                             │
│ 侧边 │       主内容区              │
│ 菜单 │       (router-view)         │
│ 200px│                             │
│      │                             │
├──────┴─────────────────────────────┤
│         Footer (可选)              │
└────────────────────────────────────┘
```

- 侧边栏宽度：200px (折叠时64px)
- 顶栏高度：60px
- 内容区最小宽度：1200px
- 表格操作列宽度：200px (固定在右侧)

## C端商城布局

```
┌────────────────────────────────────┐
│  Logo   导航栏   搜索框   购物车   │  Header
├────────────────────────────────────┤
│                                    │
│          主内容区                  │  (最大宽度1200px居中)
│                                    │
├────────────────────────────────────┤
│  Footer (版权/链接/客服)            │
└────────────────────────────────────┘
```

- 内容最大宽度：1200px 居中
- 顶栏高度：64px

## 响应式断点
| 断点 | 宽度 | 布局调整 |
|------|------|---------|
| xs | < 768px | 隐藏侧边栏，表格横向滚动 |
| sm | 768-992px | 侧边栏折叠 |
| md | 992-1200px | 正常 |
| lg | 1200-1400px | 正常 |
| xl | > 1400px | 内容区扩展 |
