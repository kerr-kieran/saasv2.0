# 组件规范

|------|------|------|
| SearchForm | components/common/SearchForm.vue | 通用搜索表单 |

| ImageUpload | components/common/ImageUpload.vue | 图片上传 |
| Editor | components/common/Editor.vue | 富文本编辑器 |
| ConfirmDialog | components/common/ConfirmDialog.vue | 确认对话框 |

### 布局组件 (layout/)

## 组件规范
- 组件名使用 PascalCase
- Props 定义使用 type + default
- 事件使用 `defineEmits`
- 所有组件必须有 `name` 属性
- 优先使用 Composition API (`<script setup>`)
