# 功能清单

## P0 (MVP — 第一版必须)

### 用户模块
- [ ] 用户注册（手机号+验证码 / 邮箱）
- [ ] 用户登录（JWT token）
- [ ] 用户登出
- [ ] Token 刷新

- [ ] 商品分类展示（树形结构）
- [ ] 商品列表（分页、筛选、排序）
- [ ] 商品搜索（关键词）
- [ ] 商品详情

### 订单模块
- [ ] 订单创建
- [ ] 订单支付（模拟/对接第三方）
- [ ] 订单取消
- [ ] 订单列表查询
- [ ] 订单详情查询

### 库存模块
- [ ] 库存查询
- [ ] 库存扣减
- [ ] 库存释放
- [ ] 库存流水记录

### 会员模块
- [ ] 个人信息查看/编辑
- [ ] 收货地址管理（CRUD）
- [ ] 积分查询

### 后台管理
- [ ] 商品管理（CRUD）
- [ ] 订单管理（列表、详情、状态变更）
- [ ] 会员管理（列表、详情）
- [ ] 库存管理（查看、调整）

- [ ] 统一路由转发

- [ ] 基础限流
