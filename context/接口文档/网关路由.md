# 网关路由

## 路由规则

| Route ID | Path | Target Service | 鉴权 |
|----------|------|---------------|------|
| user-route | /api/user/** | saas-user-service | 部分 |
| product-route | /api/product/** | saas-product-service | 否(读) |
| order-route | /api/order/** | saas-order-service | 是 |
| inventory-route | /api/inventory/** | saas-inventory-service | 内部 |
| member-route | /api/member/** | saas-member-service | 是 |
| payment-route | /api/payment/** | saas-payment-service | 部分 |
| admin-route | /api/admin/** | saas-admin-service | 是(管理员) |

## 鉴权白名单

```
POST /api/user/register
POST /api/user/login
POST /api/payment/callback/*
GET /api/product/**
```

## 限流配置

| 路径 | 限流策略 | QPS |
|------|---------|-----|
| /api/user/login | IP限流 | 10/s |
| /api/user/register | IP限流 | 5/s |
| /api/product/** | 不限流 | - |
| /api/order/** | 用户限流 | 20/s |
| /api/api/** (全部) | 总限流 | 500/s |
