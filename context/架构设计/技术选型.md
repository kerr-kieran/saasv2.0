# 技术选型

## 选型决策记录

| 领域 | 方案A | 方案B | 选择 | 理由 |
|------|-------|-------|------|------|
| 后端语言 | Java 17 | Java 11 | Java 17 | LTS版本，更好的GC、record类型 |
| 框架 | Spring Boot 2.7.x | Spring Boot 3.x | Spring Boot 2.7.x | Spring Cloud 2021.0.x 兼容性最稳定 |
| 微服务框架 | Spring Cloud 2021.0.x | Spring Cloud Alibaba 2021.x | Spring Cloud Alibaba | Nacos/Sentinel 深度集成 |
| 注册中心 | Nacos | Eureka | Nacos | 同时支持CP+AP，内置配置中心 |
| 配置中心 | Nacos | Apollo | Nacos | 与注册中心统一，减少组件 |
| 网关 | Spring Cloud Gateway | Zuul | Spring Cloud Gateway | 响应式非阻塞，性能更好 |
| ORM | MyBatis-Plus | JPA | MyBatis-Plus | 灵活SQL控制，代码生成器 |
| 数据库 | MySQL 8.0 | PostgreSQL | MySQL 8.0 | 团队熟悉度，生态成熟 |
| 缓存 | Redis 7 | Caffeine | Redis 7 | 分布式缓存，集群高可用 |
| 消息队列 | RocketMQ 5.1 | Kafka | RocketMQ | 阿里生态，事务消息支持 |
| 熔断降级 | Sentinel | Hystrix | Sentinel | Hystrix已停维，Sentinel是阿里首选 |
| 前端框架 | Vue 3 | React | Vue 3 | 团队熟悉，Element Plus生态 |
| UI库 | Element Plus | Ant Design Vue | Element Plus | 社区活跃，组件丰富 |
| 构建工具(前端) | Vite | Webpack | Vite | 开发体验好，HMR极快 |
| 构建工具(后端) | Maven 3.9 | Gradle | Maven | 团队熟悉，插件丰富 |
| CI/CD | Jenkins | GitHub Actions | Jenkins | 私有化部署，灵活度高 |
| 容器化 | Docker + Compose | K8s | Docker Compose | v1阶段简单够用，后续可迁K8s |
