# 非功能需求

## 性能指标

| 指标 | 目标值 | 说明 |
|------|--------|------|
| QPS (单服务) | 500+ | 单个微服务处理能力 |
| P99 延迟 | < 200ms | 99%的请求在200ms内响应 |
| P95 延迟 | < 100ms | 95%的请求在100ms内响应 |
| 页面加载 | < 2s | 首屏加载时间 |
| 并发用户 | 1000+ | 同时在线用户数 |
| 数据库连接池 | 20/服务 | HikariCP 默认配置 |

## 可用性

| 指标 | 目标值 | 说明 |
|------|--------|------|
| 整体可用性 | 99.9% | 年停机时间 < 8.76小时 |
| 服务容错 | N+1冗余 | 每个服务至少2个实例 |
| 故障恢复 | < 5min | 自动检测+自动重启 |
| 数据备份 | 每日全量 + 实时binlog | MySQL主从+定时备份 |

## 安全性

| 要求 | 实现方式 |
|------|---------|
| 接口鉴权 | JWT token + Gateway统一校验 |
| 数据传输加密 | HTTPS/TLS 1.3 |
| 敏感数据加密 | 密码BCrypt，手机号AES |
| SQL注入防护 | MyBatis参数化查询 (#{}) |
| XSS防护 | 前端输出编码，CSP头 |
| CSRF防护 | Token验证 |
| 权限控制 | RBAC (角色-权限-用户) |
| 日志审计 | 操作日志记录，敏感操作留痕 |

## 可扩展性

- **水平扩展**：无状态服务 + Nacos服务发现，支持动态扩缩容
- **数据库扩展**：读写分离(主从) → 分库分表(按租户ID/时间)
- **缓存扩展**：Redis Cluster 支持水平扩容
- **消息队列**：RocketMQ 支持分区扩展

## 可维护性

- 代码规范：Java (阿里巴巴规范) + Vue (ESLint + Prettier)
- 日志规范：SLF4J + Logback，统一日志格式
- 健康检查：Spring Boot Actuator + /health 端点
- 链路追踪：SkyWalking (后续接入)
- 监控告警：Prometheus + Grafana (后续接入)
