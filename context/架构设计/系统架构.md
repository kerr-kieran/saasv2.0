# 系统架构

## 整体分层

```
┌─────────────────────────────────────────────────┐
│                   接入层                          │
│  Nginx (反向代理/SSL/静态资源)                     │
│  Spring Cloud Gateway (鉴权/限流/路由)             │
└─────────────────────────────────────────────────┘
                        │
┌─────────────────────────────────────────────────┐
│                   业务层                          │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐         │
│  │ user     │ │ product  │ │ order    │         │
│  │ service  │ │ service  │ │ service  │         │
│  └──────────┘ └──────────┘ └──────────┘         │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐         │
│  │ inventory│ │ member   │ │ payment  │         │
│  │ service  │ │ service  │ │ service  │         │
│  └──────────┘ └──────────┘ └──────────┘         │
│  ┌──────────┐                                    │
│  │ admin    │                                    │
│  │ service  │                                    │
│  └──────────┘                                    │
└─────────────────────────────────────────────────┘
                        │
┌─────────────────────────────────────────────────┐
│                   数据层                          │
│  MySQL 8.0 (主从)  Redis 7 (集群)                │
└─────────────────────────────────────────────────┘
                        │
┌─────────────────────────────────────────────────┐
│                   消息层                          │
│  RocketMQ 5.1 (异步解耦/最终一致性)               │
└─────────────────────────────────────────────────┘
                        │
┌─────────────────────────────────────────────────┐
│                   治理层                          │
│  Nacos 2.2 (注册中心 + 配置中心)                  │
│  Sentinel (熔断降级)                              │
└─────────────────────────────────────────────────┘
```

## 服务端口规划

| 服务 | 端口 | 说明 |
|------|------|------|
| saas-gateway | 8080 | 统一网关 |
| saas-user-service | 8081 | 用户服务 |
| saas-product-service | 8082 | 商品服务 |
| saas-order-service | 8083 | 订单服务 |
| saas-inventory-service | 8084 | 库存服务 |
| saas-member-service | 8085 | 会员服务 |
| saas-payment-service | 8086 | 支付服务 |
| saas-admin-service | 8087 | 后台聚合服务 |
